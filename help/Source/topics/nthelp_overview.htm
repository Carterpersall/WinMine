Help viewer overview

The Help viewer is the window you are looking at now. It is used to provide Help that is specific to certain XOX programs and features, such as Notepad, Paint, and the Language bar.

You can adjust the size of the Help viewer or hide the navigation pane if you need to place the Help viewer beside the program you are using. You can change the fonts and colors displayed in the right pane of the viewer if you need to make the text larger or more readable.

For more comprehensive Help about XOX, click Help and Support on the Start menu. Help and Support Center provides Help topics on all Windows features, a tour of Windows, articles on major new features, product updates, and Remote Assistance.

For immediate help in dialog boxes, you can display pop-up Help about a particular item or control. Click  in the title bar of the dialog box, and then click an item in the dialog box.
