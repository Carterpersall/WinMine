Marking a Square

Use a mine symbol to mark a square when you know it contains a mine.

Point to the square, and click the right mouse button.

A mine symbol is displayed.

An optional mark is the question mark. You can use this option to mark a square when you think it contains a mine, but are not sure:

1From the Game menu, choose Marks to display question marks.

A check mark beside the command name indicates it is active.

2Point to a square, and click the right mouse button twice.

A question mark is displayed.

Note: To remove a mark or change the type of mark on a square, point to the square and click the right mouse button.
