Commands

------------------------------------------------------------------------------------------------------------------------------------------

This section contains information about the Game menu commands in Minesweeper.

New

Starts a new game of Minesweeper.

You can start a new game at any time, even in the middle of a game.

Beginner, Intermediate, or Expert

Sets the difficulty level of a new game. The higher the skill level, the larger the mine field.

Custom

Customizes board size and number of mines.

Marks

Allows you to mark squares with a second type of mark, the question mark. A check mark beside the command indicates it is active.

Color

Turns color on or off.

Note: You should turn on color only if you have a color monitor.

Best Times

Displays best times for all skill levels.

Exit

Exits Minesweeper.

You can exit at any time, even in the middle of a game.
